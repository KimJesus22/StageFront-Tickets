Design Notes

These skeleton screens use a structural blueprint approach. The bg-white/5 container coupled with a border-white/10 provides the glassmorphic shell, while the inner blocks use tonal variations of zinc (surface-variant and surface-bright) to suggest content layout. The CSS animate-pulse class creates the breathing effect, signaling to the user that the system is actively retrieving the high-fidelity assets.
